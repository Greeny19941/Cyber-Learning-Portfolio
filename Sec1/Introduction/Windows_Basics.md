# Windows Basics – TryHackMe Write-Up

## Overview

This lab provides a hands-on introduction to the Windows operating system, covering navigation, system tools, and basic security features.

The content aligns closely with prior experience working in Windows environments, so this lab served primarily as a refresher.

---

## Core Concepts

The lab covered:

- **User Permission Levels**
  - Basic
  - Standard
  - Administrator  

- **Authentication Methods**
  - Passwords  
  - PIN  
  - Biometrics  

- **System Navigation**
  - Start Menu  
  - File Explorer  
  - System Information  

These are fundamental to day-to-day interaction with Windows systems.

---

## System Tools & Features

Several built-in tools were explored:

- **Task Manager**
  - Monitor running processes  
  - View system performance  

- **Windows Updates**
  - Ensures systems remain patched and secure  

- **Application Management**
  - Installation via Microsoft Store or external sources  

---

## Security Features

Key security components within Windows include:

- **Windows Defender Firewall**
  - Profiles:
    - Domain  
    - Private  
    - Public  
  - Ability to create inbound and outbound rules  

- **Windows Security**
  - Centralised interface for system protection  

- **Custom Scanning**
  - Allows targeted malware scans  

Although enterprise environments often use external firewalls, local firewall configuration contributes to a **defence-in-depth** approach.

---

## Key Takeaways

- Windows includes multiple built-in security features  
- Firewall profiles help tailor network protection  
- System tools such as Task Manager assist with monitoring and troubleshooting  
- Local security controls still play a role alongside enterprise solutions  

---

## SOC Relevance

Understanding Windows is critical for security roles because:

- Most enterprise environments rely heavily on Windows systems  
- User permissions and authentication are central to access control  
- Built-in tools (e.g. Task Manager, Defender) assist in identifying suspicious activity  
- Firewall configurations can influence network behaviour and alerting  

---

## Reflection

This lab reinforced existing knowledge from practical experience in Windows environments. While the content was familiar, it highlighted the importance of built-in security features and their role in supporting a layered security approach.
