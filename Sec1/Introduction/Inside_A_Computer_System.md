# Inside a Computer System

## Overview

This lab provides a basic introduction to the core components of a computer system and their functions. It is aimed at beginners and aligns closely with foundational knowledge covered in certifications such as CompTIA A+.

The lab includes both theoretical explanations and an interactive exercise to reinforce understanding of hardware components.

---

## Core Components

The following key components were reviewed:

- **Motherboard** – The main circuit board connecting all components  
- **CPU (Central Processing Unit)** – Executes instructions and processes data  
- **RAM (Random Access Memory)** – Temporary memory used for active processes  
- **Storage (HDD/SSD)** – Persistent storage for the operating system and files  
- **GPU (Graphics Processing Unit)** – Handles graphical processing  
- **PSU (Power Supply Unit)** – Supplies power to all components  

---

## Interactive Exercise

The lab included an interactive task where components were identified and correctly placed on a motherboard. This helped reinforce physical layout and component relationships within a system.

---

## Boot Process Overview

A brief overview of the system startup process was covered:

- When a computer is powered on, it performs a **Power-On Self Test (POST)**  
- POST checks that essential hardware components are functioning correctly  
- Once completed, the system proceeds to load the operating system  

Although modern systems complete this process quickly, it remains a fundamental step in system initialization.

---

## Key Takeaways

- A computer system is composed of interconnected hardware components, each with a specific role  
- The motherboard acts as the central hub for communication between components  
- POST is a critical part of the boot process, ensuring hardware is operational before startup  
- This lab reinforced prior knowledge from A+ studies
